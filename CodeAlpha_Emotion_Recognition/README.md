# Emotion Recognition from Speech using Deep Learning

## 📌 Overview
A deep learning model that recognizes human emotions from speech audio using MFCC features and CNN+LSTM architecture trained on the real RAVDESS dataset.

## 🎯 Objective
Classify 8 human emotions from speech audio:
- Neutral, Calm, Happy, Sad, Angry, Fearful, Disgust, Surprised

## 🛠 Technologies Used
- Python
- TensorFlow/Keras
- Librosa
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Dataset
RAVDESS (Ryerson Audio-Visual Database of Emotional Speech and Song)
- Real audio speech files
- 8 emotion categories
- Downloaded from Kaggle

## 📈 Model Architecture & Performance
- Feature Extraction: MFCC (Mel-Frequency Cepstral Coefficients)
- Model: CNN + LSTM
- Sample Prediction Accuracy: 95% (19/20 correct)
- Emotions correctly identified: sad, calm, disgust, fearful, surprised, angry, happy

## 🚀 How to Run
1. Clone the repository
2. Install requirements: pip install librosa tensorflow scikit-learn pandas numpy matplotlib seaborn
3. Add your Kaggle API credentials
4. Run: emotion_recognition_speech.ipynb
