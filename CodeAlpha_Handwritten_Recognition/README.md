# Handwritten Character Recognition using CNN

## 📌 Overview
A deep learning model that recognizes handwritten digits using Convolutional Neural Networks trained on the MNIST dataset.

## 🎯 Objective
Classify handwritten digits (0-9) with high accuracy using CNN architecture.

## 🛠 Technologies Used
- Python
- TensorFlow/Keras
- NumPy
- Matplotlib

## 📊 Dataset
MNIST Dataset
- 60,000 training images
- 10,000 test images
- 28x28 pixel grayscale images

## 📈 Model Performance
- Algorithm: Convolutional Neural Network (CNN)
- Test Accuracy: 97%+
- Epochs: 5

## 🚀 How to Run
1. Clone the repository
2. Install requirements: pip install tensorflow numpy matplotlib
3. Run: handwritten_character_recognition.ipynb
