# CodeAlpha_Disease_Prediction
# Disease Prediction using Logistic Regression

## 📌 Overview
A machine learning model that predicts whether a tumor is malignant or benign using the Breast Cancer Wisconsin dataset from Scikit-learn.

## 🎯 Objective
Predict disease presence based on medical features using Logistic Regression classification algorithm.

## 🛠 Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Dataset
Breast Cancer Wisconsin Dataset (built-in Scikit-learn dataset)
- 569 samples
- 30 features
- Binary classification: Malignant vs Benign

## 📈 Model Performance
- Algorithm: Logistic Regression
- Metrics: Accuracy, Precision, Recall, F1-Score
- Visualization: Confusion Matrix

## 🚀 How to Run
1. Clone the repository
2. Install requirements: pip install scikit-learn pandas numpy matplotlib seaborn
3. Run: python disease_prediction.py
