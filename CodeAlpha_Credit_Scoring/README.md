## 📌 Overview
A machine learning model that predicts an individual's creditworthiness using financial data and Random Forest classification algorithm.

## 🎯 Objective
Predict whether an individual is creditworthy or not based on financial features like income, debt, payment history and more.

## 🛠 Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Dataset
Synthetic credit dataset with 1000 samples and 10 financial features including:
- Income
- Debt
- Payment History
- Credit Age
- Number of Accounts
- Missed Payments
- Credit Limit
- Loan Amount
- Employment Years
- Monthly Expenses

## 📈 Model Performance
- Algorithm: Random Forest Classifier
- Accuracy: 94.5%
- Precision: 93.2%
- Recall: 94.3%
- F1 Score: 93.7%
- ROC-AUC: 94.4%

## 🚀 How to Run
1. Clone the repository
2. Install requirements: pip install scikit-learn pandas numpy matplotlib seaborn
3. Run: credit_scoring_model.ipynb
